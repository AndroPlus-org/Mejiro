Mejiro

WebMejiro は第一水準漢字＋第二水準漢字＋記号＋半角全角英数字＋ひらがな＋カタカナ＋αを含んでいるウェブフォントです。

Azure Static Web Apps を使用しているため予告なく停止する場合があります。

Regular 

@font-face {
	font-family: "WebMejiro";
	src: url('https://web.mejiro.andro.plus/fonts/Mejiro-Regular.woff2') format('woff2'),
	url('https://web.mejiro.andro.plus/fonts/Mejiro-Regular.woff') format('woff'),
	url('https://web.mejiro.andro.plus/fonts/Mejiro-Regular.ttf') format('truetype');
	font-weight: normal;
	font-style: normal;
}
